📋 Análisis de Fuga (Churn) de Clientes Telco - README del Proyecto
Este proyecto realiza un Análisis Exploratorio de Datos (EDA) exhaustivo y una limpieza de datos sobre el dataset de IBM 'Telco Customer Churn' para identificar los factores clave que influyen en la pérdida de clientes.

🚀 Descripción del Proyecto
El objetivo es analizar el comportamiento y la demografía de los clientes para desarrollar estrategias de retención. El proyecto se enfoca en la limpieza de datos, el manejo de valores nulos en TotalCharges, ingeniería de características y visualización de correlaciones.

🎨 Diseño y Estética
Este notebook cuenta con una identidad visual personalizada inspirada en Discord y Twitch (#9146FF).

Nota: Se utilizó asistencia de Inteligencia Artificial para el diseño de la interfaz, los ajustes de CSS en los bloques HTML y la redacción de las anotaciones técnicas para asegurar una presentación profesional y coherente.

🛠️ Configuración Local (VS Code)
Si deseas ejecutar este proyecto de forma local en lugar de Google Colab, sigue estos pasos:

1. Prerrequisitos
Asegúrate de tener instalado Python 3.8+ en tu sistema.

2. Clonar el Repositorio
git clone https://github.com/Kuoz2/Examen_de_certificaci-n.git
cd Examen_de_certificaci-n
3. Crear un Entorno Virtual
# Windows
python -m venv venv
.\\venv\\Scripts\\activate

# macOS/Linux
python3 -m venv venv
source venv/bin/activate
4. Instalar Dependencias
pip install pandas numpy matplotlib seaborn
5. Dataset
Coloca el archivo Telco-Customer-Churn.csv en la raíz del proyecto.

📊 Características Principales
Identidad Visual: Tema oscuro con paleta púrpura (#9146FF).
Limpieza de Datos: Imputación de nulos ocultos basándose en lógica de negocio (tenure y MonthlyCharges).
EDA: Análisis profundo de contratos, antigüedad y cargos.
Codificación: One-Hot Encoding para análisis de correlación.
